# To-Do List (AWS Full CRUD Version)

A full-stack To-Do List web application deployed on AWS.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Task status & priority
- Real-time CRUD operations with AWS backend

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript (Fetch API)
- AWS Amplify Hosting

### Backend
- AWS API Gateway
- AWS Lambda

### Database
- Amazon DynamoDB

## Architecture
- The frontend is hosted on AWS Amplify
- API requests are handled by AWS API Gateway
- Business logic is implemented with AWS Lambda
- Task data is stored in Amazon DynamoDB
